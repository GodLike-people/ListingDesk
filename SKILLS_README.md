# Codex Skills

Personal Codex skills for Amazon listing and A+ content workflows.

## Included skills

- `amazon-aplus-content`
- `amazon-main-images`
- `amazon-title-bullets`

## Install on another computer

Copy each skill folder into that computer's Codex skills directory:

```text
~/.codex/skills/
```

On Windows, this is usually:

```text
C:\Users\<username>\.codex\skills\
```

Restart Codex after copying or pulling updates.
